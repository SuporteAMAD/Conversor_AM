## Integração com main.py

Adicionar no início de `main.py` (após imports):

```python
from queue_manager import QueueManager
from worker import init_worker, start_worker, stop_worker
from websocket_manager import get_ws_manager
from rotas.queue_routes import register_queue_routes

# Inicializar componentes de fila
queue_manager = QueueManager()
ws_manager = get_ws_manager()

# Inicializar worker (será startado no init)
def create_conversion_func():
    """Factory para criar função de conversão que atualiza progresso."""
    def convert_with_progress(task_id, task):
        try:
            # Lógica de conversão aqui
            # Chamar ws_manager.send_progress() durante a conversão
            # Chamar ws_manager.send_completion() ao finalizar
            pass
        except Exception as e:
            ws_manager.send_completion(task_id, False, error=str(e))
    return convert_with_progress

worker = init_worker(
    conversion_func=create_conversion_func(),
    queue_manager=queue_manager,
    max_concurrent=2  # Ajustar conforme CPU/RAM
)
```

## WebSocket (Opcional - Próximo)

Para progresso em tempo real, adicionar suporte a WebSocket:

```javascript
// Cliente
const socket = new WebSocket('ws://localhost:5000/ws');

// Subscribe ao progresso de uma tarefa
socket.send(JSON.stringify({
  type: 'subscribe',
  task_id: '< task_id_aqui >'
}));

// Receber atualizações
socket.onmessage = (event) => {
  const data = JSON.parse(event.data);
  
  if (data.type === 'progress') {
    console.log(`${data.progress}%`);
  } else if (data.type === 'completion') {
    console.log(data.success ? 'OK' : `Erro: ${data.error}`);
  }
};
```

## Inicialização da Aplicação

No startup do Flask (no app.py ou main.py):

```python
@app.before_first_request
def init_background_tasks():
    """Inicializa tasks em background."""
    start_worker()
    logger.info("Worker iniciado")

@app.teardown_appcontext
def shutdown_background_tasks(exception=None):
    """Para tasks em background ao desligar."""
    stop_worker()
    logger.info("Worker parado")
```
